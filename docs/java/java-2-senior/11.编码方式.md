

## 十一、编码方式

### 1）ASCII

计算机最初只支持英文，最先出现的字符编码是：ASCII码【采用一个字节编码】。

字母 对应的数字

'a' --> 97

 'A' --> 65

 '0' --> 48

### 2）Unicode

后来出现了一种编码方式统一了全球所有的文字，容量较大，这种编码方式叫做：Unicode编码。

### 3）UTF-8

8-bit Unicode Transformation Format

### 4）GBK、GB2312

支持简体中文的编码方式：
 GB2312 < GBK < GB18030 按照容量大小

### 5）如何解决乱码问题

编译时，指定用编写时使用的编码
